# 🐍 Snake Game · 8086 Assembly

> Classic Snake game – eat food (`*`), grow longer, don't crash.  
> Written in **pure 8086 Assembly** using only BIOS/DOS interrupts.

---

## 🎮 Features

| Feature | Description |
|---------|-------------|
| 🎯 **Real‑time input** | Arrow keys, non‑blocking |
| 🧠 **Direction buffer** | No 180° instant reversal |
| 🍎 **Random food** | Timer‑based pseudo‑random |
| 📊 **Score display** | Updates live |
| 💥 **Collision** | Walls + self‑body |
| 🔁 **Restart** | Press `R` after game over |

---

## 🖥️ How to Run

1. Open **emu8086**
2. Load `snake.asm`
3. Click **Emulate** (or `F5`)
4. Play with `↑ ↓ ← →`
5. Game over? Press `R` to restart, `Esc` to quit
