# -CEN323_G15_-Classic-snake-Game-
Classic Snake game written in 8086 Assembly (emu8086).  Control the snake (O/o) to eat food (*), grow, and avoid walls + yourself. Demonstrates real‑time input, array manipulation, collision detection, and BIOS/DOS interrupts. Modular code with 15+ procedures. Press R to restart after game over. Developed for CEN 323 – COAL semester project.
